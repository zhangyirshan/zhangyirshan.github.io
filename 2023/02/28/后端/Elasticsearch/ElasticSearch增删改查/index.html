创建索引

使用PUT请求创建索引，PUT操作有幂等性，重复请求会报错

PUT http://127.0.0.1:9200/shopping

创建文档

使用POST、PUT请求创建索引文档，可重复请求，使用_create不可以重复请求，其余情况可以重复请求

POST http://127.0.0.1:9200/shopping/_doc 可以重复
POST http://127.0.0.1:9200/shopping/_create 不可以重复
PUT http://127.0.0.1:9200/shopping/_doc 可以重复
PUT http://127.0.0.1:9200/shopping/_create  不可以重复
指定id
POST http://127.0.0.1:9200/shopping/_doc/1003 可以重复
PUT http://127.0.0.1:9200/shopping/_create/1003 不可以重复

条件查询

match=>where
match_all=> 查询所有
from=》limit 起始点从0开始
size=》limit 步长
sort=》order
aggs=》聚合函数

must=》and
should=》or

filter:{
range:{
price:{
gt: 5000
}
}
}

GET http://127.0.0.1:9200/shopping/_search


{
    "query":{
        "match":{
            "category":"小米"
        }
    }
}

{
    "query":{
        "match_all":{
        }
    },
    "from": 0,
    "size": 2,
    "_source": ["title"],
    "sort": {
        "price": {
            "order": "desc"
        }
    }
}

match分词匹配，输入小米，会将含有小和米的词都查出来。
match_phrase全匹配，必须全部符合才可以查询出来，输入小米，必须等于小米
高亮显示的属性填写到fields下

{
    "query": {
        "match": {
            "category": "小花"
        }
    },
    "highlight": {
        "fields": {
            "category": {}
        }
    }
}
