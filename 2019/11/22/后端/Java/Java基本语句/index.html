3、break语句与continue语句

  1. break 语句： 经常用在循环语句中，用于跳出整个循环，执行循环后面的代码。
  2. continue 语句：经常用在循环语句中，用于跳出当前的这个循环（或者是跳出本次循环），开始下一次循环的执行。
  3. break 与 continue 可以搭配标签使用， 在实际开发中，根本没有人会将 break 与 continue搭配标签来使用。
