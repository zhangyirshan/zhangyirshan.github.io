角色

  • 发布者（publisher）
  • 订阅者（subscriber）
  • 频道（channel）

模型

没有消息堆积的功能，无法查询历史发布消息

API

publish（发布命令）

subcribe（订阅）

unsubcribe（取消订阅）

其他API

  1. osubscribe [pattern …] # 订阅模式
  2. punsubscribe [pattern …] # 退订指定的模式
  3. pubsub channels # 列出至少有一个订阅者的频道
  4. pubsub numsub [channel …] # 列出给定频道的订阅者数量
  5. pubsub numpat # 列出被订阅模式的数量
