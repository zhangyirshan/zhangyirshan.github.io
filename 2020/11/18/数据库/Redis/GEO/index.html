GEO

GEO(地理信息定位)：存经纬度，计算两地距离，范围计算等

例：

北京：116.28，39.55
天津：117.12，39.08

应用场景

API

geoadd

geopos

geodist

georadius

相关说明

  1. since 3.2+
  2. type geoKey = zset
  3. 没有删除API：zrem key member
