创建索引轮循器

Name：名称

Hot phase

先选择是否开启rollover
三个max共同限制回滚界限

Delect phase

创建索引模板

Logistics

Name: 索引模板的名称
Index patterns：适用于哪些索引例如：rongyun-*适用于所有以rongyun-开头的索引

Index settings

lifecycle.name: 轮循器的名称
lifecycle.rollover_alias: 索引别名，用于轮循器迭代标记索引时使用
refresh_interval: 索引刷新时间
