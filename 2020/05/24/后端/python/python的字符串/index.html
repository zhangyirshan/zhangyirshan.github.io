原始字符串

原始字符串的使用非常简单，只需要在字符串前加一个字母r即可，注意，原始字符串不能在结尾放\会报错

解决办法

  1. 在后面拼接str = r'C:\new\aaa\bbb'+'\\'
  2. 在后面加上[:-1] str = r'C:\new\aaa\bbb\\'[:-1]
